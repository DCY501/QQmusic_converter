# 🎵 QQ音乐OGG转MP3转换器

将QQ音乐下载的OGG格式文件转换为标准MP3格式，支持批量转换和音质选择。

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

## ✨ 功能特点

- 🚀 批量转换：支持同时转换多个OGG文件
- 🎚️ 音质选择：VBR高质量 / CBR 320k / CBR 192k
- 💻 图形界面：简单易用的GUI，无需命令行
- 💾 配置记忆：自动保存上次的输出目录和音质设置
- 🔒 本地处理：文件在本地转换，不上传服务器

## 📦 下载使用

###  直接下载EXE
前往 [Releases](https://github.com/DCY501/QQMusic_converter/releases) 页面下载最新版本：
- `QQ音乐转换器.exe` - 开箱即用，无需安装Python
